# Laboration 2 - ECMAScript

En laboration inom ECMAScript som syftar till att hämta in extern data genom API-anrop async & await samt try & catch. Sortering och filtrering av data genom sökfält och klick på rubriker som uppdateras automatiskt. Automatisering genom VITE och kommandot npm run build. 

Webbplats: https://lab2frontend-fb.netlify.app/
